Summary
-------

Views order form turns a view into an order form.

Requirements
------------

Views http://drupal.org/project/views


Installation
------------

Install as usual, see http://drupal.org/node/70151 for further information.


Configuration
-------------

1. Configure the module at admin/structure/views/settings/viewsorderform

2. Create a view and add a 'Views order form' field.
